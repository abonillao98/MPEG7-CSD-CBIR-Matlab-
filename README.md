# MPEG7-CSD-CBIR-Matlab

A Content-Based Image Retrieval system using the CSD color descriptor from the MPEG-7 standard in Matlab.

This is an academic project developed for the **"Image and Video Processing"** subject, as part of the **"Audiovisual Systems"** specialisation of the **Bachelor's Degree in Telecommunications Technologies and Services** at the UPC (Telecos BCN – ETSETB).

It was developed during the **Spring semester of the 2024–2025 academic year** (from February to May 2025).

An explanation of the project can be found in the **"Memoria.pdf"** document (in Catalan).
