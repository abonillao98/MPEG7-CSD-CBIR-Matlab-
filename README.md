# MPEG7-CSD-CBIR-Matlab
A Content-Based Image Retrieval using the CSD color descriptor from the MPEG 7 standard in Matlab
This is an academic project devoloped for the "Image and Video Processing" subject, as part of the "audiovisuals systems" specialisation of the Bachelor's degree in Telecommunications Technologies and Services held by the UPC (Telecos BCN - ETSETB)
It was devoloped during the Spring semester of 2024-2025 academic year (from February to May 2025)
An explanation of the project can be found in the "Memoria.pdf" document (in Catalan language)
